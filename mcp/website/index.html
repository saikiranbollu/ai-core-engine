Getting Started

Connect to the AI Core Engine MCP server and start using tools

1

Configure your MCP Client

Add the AI Core Engine server to your MCP client configuration (e.g. VS Code Copilot):

VS Code — .vscode/mcp.json
{
  "servers": {
    "aice": {
      "type": "http",
      "url": "https://aice-mcswai.eu-de-7.icp.infineon.com/mcp",
      "headers": {
        "Authorization": "Bearer ${input:aice-api-key}"
      }
    }
  }
}

Request your API key from the team — never hardcode keys in shared files. Use input variables or environment variables as shown above.

2

Verify Connection

Once configured, ask your AI assistant to run the health check tool:

Prompt
Use the health_check tool to verify the AICE server is reachable.

Expected response: a JSON object with backend service statuses (neo4j, qdrant, redis).

3

Try Basic Queries

Here are some example prompts to get started:

Search the knowledge graph
Use search_database to find all nodes related to "CAN driver initialization"
Query an API function
Use query_api_function to get documentation for "IfxCan_Can_initModule"
Trace requirements
Use find_requirement_traces to trace requirement REQ-1234 to implementation
Upload a file for analysis
Use sandbox_upload to ingest my current file, then search_database with
the returned session_id to query its contents.
4

Explore All Tools

Visit the Docs page to browse all 56 available tools, or open the Swagger UI for the full OpenAPI specification.

Tools are organized into access tiers:

  • public Available to all authenticated users
  • developer Requires developer role
  • admin Admin-only operations

About Us

AI Core Engine Team

We are the AI Core Engine team at Infineon Technologies, building the next-generation knowledge graph platform for automotive software engineering. Our platform connects requirements, source code, test artifacts, and hardware interfaces into a unified, queryable graph — enabling intelligent traceability and AI-powered insights.

What We Do

  • Knowledge Graph: Neo4j-powered graph connecting all software artifacts
  • Hybrid RAG: Combining graph traversal with vector search for intelligent retrieval
  • MCP Server: 56 tools exposed via the Model Context Protocol for AI assistant integration
  • Review Gate: Human-in-the-loop confidence evaluation for critical decisions
  • Observability: Full Prometheus + Grafana monitoring stack
